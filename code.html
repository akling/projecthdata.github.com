hData Code Repositories

hData XML Schemas
Schemas that describe the hData root.xml document, section metadata and a continuity of care content profile.
Java hStore
A Java Implementation of the hData RESTful API that can be easily extended to integrate with existing health information systems. There is also a sample project showing how a simple, in-memory health record could be accessed.
Lab Content Profile
An hData Content Profile for lab orders and results.
HL7 TSC Pilot Project
This repository contains code that will allow you to translate hData Records into CDA documents and vice versa for the CDA R2 Tuberculosis Follow Up Progress Note. A full description of the information contained in the progress note can be found on the Structured Document Template Design Pilot Page.
Ruby hStore
A Ruby implementation of the hData RESTful network transport that stores information into MongoDB.
